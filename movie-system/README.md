# **Installation**

```sh
yarn install
```

# **Run App**

```sh
# development
yarn run start

# watch mode
yarn run start:dev

# production mode
yarn run start:prod
```

# **Test**

```sh
# unit tests
yarn run test

# e2e tests
yarn run test:e2e

# test coverage
yarn run test:cov
```

# **Build Docker Image**

```sh
docker build -t movie-system:latest .
```
